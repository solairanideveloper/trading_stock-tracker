ABSTRACT

The contemporary financial landscape is characterized by an unprecedented inflow of data, making it increasingly difficult for retail investors to distinguish between market noise and actionable signals. This project, titled "Signalist", presents an advanced AI-powered platform designed to provide intuitive and intelligent stock market tracking. Unlike traditional static dashboards, Signalist serves as a proactive intelligence layer that synthesizes real-time market data with generative AI insights.

Built upon the Next.js 15 framework, the platform utilizes high-performance server components and client-side TradingView integrations to provide millisecond-level data accuracy. The fundamental innovation lies in the orchestration of event-driven workflows using Inngest, which enables the system to perform complex asynchronous tasks such as market sentiment analysis and automated earnings summaries without compromising user experience.

The intelligence core is powered by the Google Gemini Flash model, which processes thousands of news points to generate a "Daily Digest" tailored to the specific symbols in a user's watchlist. Data persistence is handled via a cloud-native MongoDB Atlas cluster using Mongoose for object data modeling, ensuring high availability and ACID compliance across distributed transactions. For security, Better Auth provides a robust multi-strategy authentication layer that supports secure session management and cross-device sync.

Experimental results show a significant reduction in the "Time-to-Insight" for users, as the system proactively delivers context alongside data. This project demonstrates how modern full-stack development patterns and AI orchestration can be combined to solve real-world financial data transparency issues, providing a state-of-the-art tool for the next generation of digital investors.

INTRODUCTION

1.1 Background & Motivation

The global equity markets have transformed from physical trading floors to complex digital ecosystems that operate 24/7. In this transition, the primary challenge for individual investors has shifted from access to information to the intelligent processing of information. Historically, retail investors were limited by delayed ticker data and physical news reports. Today, they are overwhelmed by a deluge of real-time prices, social sentiment, and automated news feeds.

The motivation for Signalist arose from the observation that while raw data is now abundant and free (via interfaces like TradingView or Yahoo Finance), meaningful context is still locked behind expensive professional terminals like Bloomberg or Refinitiv. There exists a significant gap in the market for a "context-first" financial tool that doesn't just show a red or green candle, but explains the fundamental or narrative driver behind market moves using artificial intelligence.

By leveraging Large Language Models (LLMs) and high-speed web frameworks, Signalist aims to democratize professional-grade financial intelligence. The goal is to provide a platform that feels like having a personal financial analyst who monitors your specific stocks 24/7 and proactively provides summaries, alerts, and insights without requiring constant manual oversight from the user.

1.2 Problem Statement

Most existing financial tracking applications suffer from three critical architectural and design flaws: clutter, information overload, and high latency.

First, traditional platforms often prioritize ad revenue over user experience, leading to cluttered interfaces that distract the user. Second, these platforms provide news articles in a chronological list format with no prioritization based on user interest or symbol importance. A user with 10 stocks in their watchlist might see 200 articles a day, making it impossible to read everything. Third, many legacy systems use slow, synchronous data fetching methods that lead to poor mobile performance and high Time-to-Interactive (TTI).

Signalist addresses these by implementing a "Bento-style" minimalist UI, a multi-step AI summarization pipeline, and an asynchronous, edge-ready tech stack that ensures high speed and high relevance.

1.3 Objectives & Goals

  • Intuitive Dashboard: Create a visually stunning, responsive dashboard using a modern "Bento Box" design system.
  • High-Speed Visualization: Integrate professional-grade TradingView charting components for seamless technical analysis.
  • AI-Driven Insights: Utilize the Google Gemini Pro model to synthesize market news into short, actionable daily digests.
  • Secure Multi-Device Sync: Implement a robust authentication layer to allow users to access their personalized watchlists across various devices securely.
  • Event-Driven Proactivity: Develop a background workflow engine to monitor price triggers and market events automatedly.

1.4 Scope & Limitations

The project scope encompasses the full-stack development of the web application, including the frontend UI, backend API, database persistence, and AI orchestration. The application focuses on the US equity market (NASDAQ, NYSE) due to the availability of high-fidelity API data. Future versions could potentially expand into global markets (LSE, NSE) and other asset classes like Cryptocurrencies and Forex.

LITERATURE SURVEY

2.1 Evolution of Stock Tracking Systems

The history of stock tracking is a journey from manual ledgers to AI-driven automation. In the late 19th century, the "Stock Ticker" was the primary innovation, allowing prices to be transmitted over telegraph wires. In the 1980s, the Bloomberg Terminal revolutionized the industry by providing a centralized digital data stream. The 2010s saw the rise of cloud-based visualizers like TradingView, which brought professional oscillators and indicators to the web browser.

The current phase, which Signalist belongs to, is the **"Intelligence Native Phase."** In this phase, the system is not just a pipe for data but a brain that interprets data. Research into financial sentiment analysis has moved from simple bag-of-words models to context-aware Transformers that can understand the difference between "Apple prices falling" (bad news) and "Falling for Apple's new product" (good news).

2.2 Role of AI in Financial Analysis

Artificial Intelligence is no longer an optional feature in finance; it is a core component. Academic studies have shown that LLMs trained on financial datasets outperformed traditional quantitative models in predicting short-term sentiment shifts. By using LLMs like Gemini Flash, we can perform "Zero-Shot" summarization of earnings calls and news reports with higher accuracy than previous NLP models. This allows retail users to process 10 hours of market reading in 10 minutes of AI summaries.

4.3 UI Design Strategy: The Bento Box

The user interface follows a "Bento Grid" philosophy. In Japanese culture, a bento box organizes multiple food items into separate, clean compartments. Similarly, Signalist organizes complex financial data into discrete, visually separated cards. This design pattern reduces cognitive load by grouping related information (e.g., Heatmap in one card, News in another) and allowing the eyes to scan information efficiently.

IMPLEMENTATION & CODING

5.1 Frontend Module Implementation

The frontend is built with React 19 (within Next.js 15). We used Shadcn UI for the primitive components, customized with Tailwind CSS for the dark-mode primary aesthetic. High-performance charting is achieved through the TradingViewWidget.tsx component, which utilizes dynamic script loading to prevent render-blocking.

5.2 Backend API & Server Actions

Instead of traditional express.js routes, Signalist leverages Next.js Server Actions. This allows the frontend to call backend functions directly with full type safety. The finnhub.actions.ts file handles all upstream communication with market data providers, implementing an intelligent caching layer to minimize API credit consumption.

5.3 AI & LLM Integration (Gemini)

The AI workflow is the most complex implementation. We use Inngest to handle the retry logic and state management of long-running AI tasks. When a daily report is triggered, the system fetches news in chunks, sends them to Gemini for "Thematic Extraction," and then compiles those themes into a personalized HTML email template. This ensures that even if one AI call fails, the entire workflow is resumed from the nearest checkpoint.
