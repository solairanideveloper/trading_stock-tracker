ABSTRACT

The contemporary financial landscape is defined by an overwhelming influx of real-time data, which often leads to "Analysis Paralysis" for retail traders. This project, "Signalist," develops a sophisticated AI-native stock tracking and analytics system designed to bridge the intelligence gap between institutional and retail participants. Built with the Next.js 15 framework, Signalist provides a premium "Bento Grid" dashboard for monitoring global financial equities.

The project integrates Google Gemini Flash for automated market sentiment analysis and context-aware summarization of thousands of global news headlines. By utilizing the Inngest event-driven workflow engine, Signalist proactively delivers synthesized market intelligence, effectively automating the role of a professional research analyst. The platform achieves industrial benchmarks in performance, security (Better Auth), and cost-efficiency, providing a blueprint for the next generation of AI-driven financial software.

1. INTRODUCTION

1.1 Project Overview

Signalist is an "Intelligence-First" financial tracking platform. Unlike traditional stock trackers that merely display price charts, Signalist synthesizes global market narratives into actionable bulletins using Artificial Intelligence. Its core focus is on reducing the cognitive load on investors by filtering out the "Market Noise" and amplifying the "Investment Signal."

Built using Next.js 15, the platform is optimized for the serverless edge, ensuring that data is always fresh and interaction is always smooth across all devices.

1.2 Motivation

The project was motivated by the observation that retail investors are increasingly participating in the markets without the analytical tools necessary to understand the "why" behind price movements. Signalist aims to democratize access to high-quality financial synthesis using Large Language Models (LLMs).

1.3 Objectives

The primary objectives of this project are: First, to develop a zero-latency financial dashboard. Second, to automate market news summarization. Third, to deliver proactive market insights via durable background workers.

2. LITERATURE SURVEY

The history of financial tracking has moved from paper ticker tapes to electronic terminals to cluttered web portals. However, we are currently entering the "AI-Analyst Era." Literature survey shows that traditional sentiment analysis (based on keyword matching) is inferior to modern LLMs (based on Transformer architecture) which understand corporate context, regulatory nuances, and geopolitical risks.

By comparing Signalist with legacy platforms like Yahoo Finance and Morningstar, we identified a critical lack of "Narrative Automation"—a gap that this project successfully fills using the Gemini ecosystem.

3. SYSTEM ANALYSIS

System analysis confirmed that a serverless architecture is the most cost-effective and scalable approach for a real-time tracking application. Technical feasibility was established by integrating Finnhub's institutional APIs with Next.js's server actions. Requirements gathering specified that the system must handle over 10,000 tickers and deliver AI summaries with less than 500ms latency on the edge.

4. SYSTEM DESIGN

Signalist follows a Modern Three-Tier Serverless Architecture. The UI is composed of high-performance React components. The Logic Tier uses Next.js Server Actions for secure database mutations. The Intelligence Tier utilizes Inngest workflows to orchestrate Gemini AI calls, ensuring that complex summarization tasks are completed reliably in the background without blocking the user interface.

Database modeling in MongoDB uses a Document-based approach for flexibility in tracking various financial metrics and user preferences.

The results are stored and delivered to the user, providing a reactive feedback loop in their investment journey.
