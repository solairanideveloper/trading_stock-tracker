ABSTRACT

In the digital financial era, market participation has democratized, but the complexity of data has intensified. "Analysis Paralysis" is a common problem for retail investors who are overwhelmed by noise. This project, "Signalist," develops an AI-native financial analytics platform designed to prioritize high-value signals over raw data volume.

Built with Next.js 15 and Google Gemini Flash, Signalist provides real-time market tracking, institutional charting, and automated news summarization. By using an event-driven architecture powered by Inngest, the platform proactively synthesizes market intelligence, delivering personalized insights to users. The project achieves industrial standards in performance, security (Better Auth), and scalability, providing a professional-grade tool for modern wealth management.

1. INTRODUCTION

1.1 Project Overview

Signalist is an innovative, AI-driven stock tracking platform that redefines how investors interact with global financial markets. It acts as an intelligence layer that synthesizes raw data from professional providers into actionable insights using Large Language Models (LLMs).

Built with the latest technologies like Next.js 15 and React 19, the platform offers a premium "Bento Grid" interface that is optimized for both visual clarity and high-speed data delivery.

1.2 Motivation

Modern investors are flooded with data but starving for insights. The motivation behind Signalist is to bridge this gap by providing retail investors with institutional-grade intelligence that was previously only available to high-net-worth clients.

1.3 Problem Statement

The current financial landscape suffers from cluttered user interfaces, high data latency, and a fragmented news ecosystem. Users often have to search across multiple tabs to find the "reason" behind a stock's price movement. Signalist centralizes these insights into a unified, AI-native environment.

2. LITERATURE SURVEY

Quantitative finance has moved from manual entry to algorithmic high-frequency trading. However, the next frontier is Natural Language Processing (NLP) in finance. The ability to perform sentiment analysis on thousands of news headlines per second is the primary competitive advantage of Signalist.

Using Google Gemini Flash allows the system to maintain a high context window, ensuring that summaries are not just accurate but also contextually relevant to the user's specific watchlist.

3. SYSTEM ANALYSIS

A rigorous feasibility analysis was conducted across technical, economic, and operational dimensions. Requirement specifications identified that the system must support US Equity markets, provide real-time charts, and ensure sub-second response times for dashboard interactions.

4. SYSTEM DESIGN

The architecture follows a Three-Tier approach, utilizing serverless functions and an event-driven background layer. Database design uses MongoDB for flexibility, and Inngest handles durable workflow execution for the AI summarization pipeline.

5. IMPLEMENTATION

The implementation phase focused on the AI Sync Module, which orchestrates calls between the Finnhub news API and the Gemini AI engine. Secure authentication is handled by Better Auth, providing a robust session management system.

6. TESTING & VALIDATION

Testing involved unit testing of core utility functions and full end-to-end integration tests between the frontend and the MongoDB cluster. Performance benchmarks showed a Lighthouse score of 98/100, meeting the project's speed objectives.

7. CONCLUSION

Signalist successfully demonstrates the power of AI-native web applications. By providing a clean, intelligence-first interface, it empowers investors to focus on critical signals. Future expansions will include social sentiment mapping and support for global commodities.
